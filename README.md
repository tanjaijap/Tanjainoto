## ก่อน Deploy — สิ่งที่ต้องทำ

1. **optimize รูปภาพ** — แยกรูปออกจาก HTML ใส่ใน /images/
2. **เปลี่ยน email** — hello@tanjainoto.com → email จริง
3. **เปลี่ยน LINE** — @tanjainoto → ID จริง
4. **เปลี่ยนเลข PromptPay** — ใส่เลขจริง
5. **ผูก domain** — tanjainoto.com หรือ .co

## วิธี Deploy (Netlify)

1. ไปที่ app.netlify.com/drop
2. ลาก folder tanjainoto-site วางลงหน้านั้น
3. ได้ลิงก์ทันที!
